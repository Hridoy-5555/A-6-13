# DigiTools - AI Marketplace

DigiTools is a premium digital marketplace designed for creators and developers. It offers a curated selection of AI-powered writing tools, design asset bundles, SEO audit software, and productivity utilities to supercharge your digital workflow.

## 🚀 Technologies Used
- **React.js**: Frontend framework for building the interactive UI.
- **Tailwind CSS & DaisyUI**: For rapid, responsive, and beautiful component styling.
- **JavaScript (ES6+)**: Functional logic and state management.
- **React-Toastify**: Professional notification system for user interactions.
- **JSON**: Mock database for product information.

## ✨ Key Features
1.  **Dynamic Product Ecosystem**: Browse a variety of digital tools with real-time filtering (Buy Now state) and detailed feature lists.
2.  **Seamless Cart Management**: Add products to your cart with instant UI updates, view total costs, and manage selections with a single-column checkout interface.
3.  **Interactive User Experience**: Includes a togglable view between the product gallery and the shopping cart, integrated with `react-toastify` for instant feedback on every action.

## 🛠️ Installation & Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` or `npm run dev` to launch the application.
